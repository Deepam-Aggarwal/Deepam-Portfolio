Project Info:

Let's understand it with the help of an situation:
As an employee at Extreme Car Transport, I want to ensure I can use this application in order to fulfill the purpose of checking confirmations, notifications, tracking and reviews.
As an employee, I want to be able to check bookings, edit the booking calendar as well as delete any cars that are no longer booked.
As an employee, I want to be able to manage the reviews shared on social media via this application.
As an employee, I want to be able to review all of the cars in motion, the miles as well as the speed of the car in motion
As an employee, I want to have a simple menu where I can select all of the functionality I need to complete my job on a daily basis.
